Les éléments de structure permettent d'organiser les données dans un portfolio.

Dans la représentation XML du portfolio, les éléments de structure composent les branches de l'arbre, alors que les éléments de données (les ressources) en sont les feuilles.

Les éléments de structure Karuta sont :

  • Section
  • Page
  • Sous-section

Les éléments de données Karuta sont :

  • Nœud ressource

Section

Un élément Section permet de regrouper des pages du portfolio. Une Section peut contenir d'autres éléments Section, des éléments Page, des éléments Sous-section et des Nœuds ressource.

Page

Un élément Page est l'unité d'affichage d'un portfolio. Une page peut contenir des éléments Sous-section et des Nœuds ressource.

Sous-section

Un élément Sous-section permet de regrouper des ressources. Une Sous-section peut contenir d'autres éléments Sous-sections et des Nœuds ressource.

Nœud ressource

Un Nœud ressource est l'élément terminal d'un arbre (les feuilles). Une ressource n'est pas un élément de structure, elle contient les données du portfolio. Il existe plusieurs types de ressources : champ de texte, éditeur de texte, fichier, image, audio, vidéo, calendrier, etc (voir la page RESSOURCES pour plus de détails).
